# Commute Dashboard HK

A personal mobile web dashboard for my morning commute in Hong Kong.

## Goal
Help me quickly decide each morning: **Can I still take the bus?**

## Main inputs
- Bus ETA for my saved commute routes
- Traffic condition between home and work
- Weather at home and work
- Commute-related alerts, especially road accidents

## Target platform
- iPhone browser

## V1 scope
- Preferred bus route
- Backup bus route
- Train fallback
- Manual setup for route and stop IDs
- Mobile-first dashboard
- Simple verdict at the top

## Not in V1
- Route optimization
- Public route search
- GTFS integration
- Maps
- User accounts
- Notifications
- General news feed

## Why this project
I currently need to check multiple apps every morning. This dashboard is meant to bring the key commute signals into one simple screen.
